# 下载MySQL

  • 这里下载的离线完整包
  • 前往官网下载：点此直达open in new window
  • 下载社区免费版，选择上方Windows (x86, 64-bit), ZIP Archive版本，下方MySQL开发使用。

# 选定IDE

  • 选择一个数据库的IDE，DataGrip,SQL Server,navicat等等等等，任选其一即可
  • 这里推荐DataGrip，很方便的可以可视化处理数据库的工具。如果是大学生，可凭借学信网信息拿到他们家的教育许可证，一系列软件可以免费使用

# JDBC连接器

  • 可下载，点此直达open in new window

  • 操作系统选择Platform Independent(平台无关)。即可显示链接

  • 若读者使用的DataGrip，可在该IDE设置界面的时候下载相关连接器驱动。

# 安装MySQL

以下为离线完整包初始化，

  1. 解压后先将MySQL的bin目录添加到环境变量，以便使用。

  2. 以管理员权限运行终端窗口， 进行数据库初始化

    mysqld --initialize --console
    1

    此时在解压根目录下多出一个data文件夹，其中.err文件包含root最高权限的临时密码

    或者在执行后控制台输出的信息中的 root@localhost: 后的符号字母数字随机组合即为临时密码。

  3. 输入以下代码安装sql。

    mysqld install
    1

  4. 启动sql服务

    net start mysql
    1

  5. 更改密码

    mysqladmin -u root -p password
    1

  6. 最后用更改好的密码进行测试

     mysql -u root -p
    1
  • 显示 mysql> 说明可以使用了

# 加载连接器驱动

  • DataGrip用户在打开该IDE后，加载数据源时，IDE会提示缺少驱动，一步步来就可以。

  • 若下载失败，在加载数据源的界面找到驱动程序栏，手动添加 jar包 （不是zip压缩包） open in new window

  • 项目添加JDBC的连接器：前往项目所在的文件夹，找到lib文件夹，没有就新建，将连接器解压后的 jar包 整个移动到其中，在项目设置中将其中的jar包添加为依赖，即可使用。 open in new window

  • 若使用的eclipse,在执行相同的操作后，右键 jar包 ，选择Build Path将其添加为项目依赖。 open in new window

  • 最后，进行数据库的链接查看有没有报错。
