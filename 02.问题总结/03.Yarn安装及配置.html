# yarn 下载 安装

  • yarn 1：请点击这里open in new window
  • yarn 2：请点击这里open in new window

# 配置(以下适用于yarn 1)

  1. 查看版本，正常显示为安装成功

    yarn -v
    1

  2. 修改yarn全局安装目录

    yarn config  set global-folder "修改的目录位置"
    1

  3. 修改yarn缓存位置

    yarn config set cache-folder "修改的目录位置"
    1

    在添加全局包的时候，会在全局安装目录下生成node_modules/.bin目录。 此时需要将该目录的路径添加到系统环境变量中，否则即使通过yarn global 添加到全局包在命令行中也是找不到的。

  4. 检查当前bin目录的位置

    yarn global bin
    1

  5. 检查当前全局安装的位置

    yarn global dir
    1

  6. 查看yarn配置

    yarn config list
    1
